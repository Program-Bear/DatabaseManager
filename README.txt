第15小组数据库阶段性汇报：
项目开始后第一周内我们主要在研究文件系统的相关接口文档，并尝试实现了记录的管理的部分功能，经过协商我们确定了数据中记录的存放的基本形式，并实现和定义Record类
在压缩包Reocrd_manager.zip中有一个文件夹Record，里面主要有以下几段代码：
1. record_manager.h -> 这一段主要用于实现文件的插入，删除，打开，关闭，记录的插入，删除，修改，查询，但是由于我们的索引模块还没有完成，所以这一部分还没办法进行测试
2. page_manager.h -> 这是我们下一个开发周期打算集中开发的页管理系统，能支持记录在页内的一系列操作，这个部分的代码还没有完成
3. Record.h ->这个头文件定义了对数据库中记录的具体存储方式，以及一系列相关操作，可以通过下列指令来编译获得对这部分的测试程序：
	g++ Record.cpp -o Record_test
另外我们项目的git仓库在https://github.com/Program-Bear/DatabaseManager，欢迎随时关注我们的开发进展
